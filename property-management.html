🔄 訂房 → IoT 自動化流程

📅
1. 住客完成訂房
LINE LIFF / OTA 平台 → 系統建立訂單，狀態 = CONFIRMED
🔑
2. 自動生成臨時密碼
TTLock Password Service → 6位隨機密碼 → 透過閘道器同步到門鎖（3次重試機制）
⏰ 有效期：入住前 30min ~ 退房後 30min
💡
3. 設備預排程
燈光排程：入住時段自動開啟｜電梯授權：綁定住客樓層｜空調預設：提前 15min 啟動
🚪
4. 住客入住（密碼開門觸發）
開門事件 → 系統標記「已入住」→ 客房設備全部啟動 → LINE 推播歡迎訊息
🔒
5. 退房自動回收
密碼過期自動失效 → TTLock 刪除密碼 → 燈光/空調關閉 → 電梯權限收回 → 房態 = 待清潔
🤖 全流程自動化 — 管理員零操作，系統自動處理從密碼派發到設備回收的每一步
