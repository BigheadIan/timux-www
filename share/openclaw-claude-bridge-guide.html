前置條件

環境需求

  • Claude Code CLI 已安裝並登入 Max 訂閱帳號
  • Node.js >= 18
  • OpenClaw >= 2026.1（Gateway 運行於 port 18789）
# 驗證
claude --version
claude auth status   # 確認已登入 Max 帳號

一、Bridge 安裝

# 1. Clone
git clone https://github.com/shinglokto/openclaw-claude-bridge.git
cd openclaw-claude-bridge
npm install   # 會自動 build Dashboard

# 2. 設定 .env
cp .env.example .env
# 編輯 .env，至少設定 DASHBOARD_PASS

# 3. 啟動測試
npm start
curl http://localhost:3456/health   # 應回傳 {"status":"ok"}

端口說明

端口 綁定 用途
3456 localhost OpenAI 相容 API（僅本機）
3458 0.0.0.0 Dashboard（區域網絡可存取）

重要環境變數

變數 預設 說明
DASHBOARD_PASS — Dashboard 密碼（用戶名 admin）
IDLE_TIMEOUT_MS 120000 Claude CLI 無輸出超時（ms）
MAX_PER_CHANNEL 2 每 channel 最大同時請求數
OPUS_MODEL opus Opus 的 CLI 模型別名
SONNET_MODEL sonnet Sonnet 的 CLI 模型別名

二、macOS 開機自啟（launchd）

# 一鍵安裝（自動偵測路徑、讀取 .env）
./service/install-mac.sh

手動管理

# 重啟
launchctl bootout gui/$(id -u)/com.openclaw.claude-bridge
launchctl bootstrap gui/$(id -u) ~/Library/LaunchAgents/com.openclaw.claude-bridge.plist

# 日誌
tail -f ~/openclaw-claude-bridge/bridge.log
重要：.env 改動後要重跑 ./service/install-mac.sh

三、OpenClaw 設定

在 ~/.openclaw/openclaw.json 新增 provider：

{
  "models": {
    "providers": {
      "claude-bridge": {
        "baseUrl": "http://localhost:3456/v1",
        "apiKey": "not-needed",
        "api": "openai-completions",
        "models": [
          {
            "id": "claude-opus-latest",
            "name": "Claude Opus",
            "contextWindow": 1000000,
            "maxTokens": 128000,
            "reasoning": true
          },
          {
            "id": "claude-sonnet-latest",
            "name": "Claude Sonnet",
            "contextWindow": 1000000,
            "maxTokens": 64000,
            "reasoning": true
          }
        ]
      }
    }
  }
}
模型指定：將 agent 的模型設為 claude-bridge/claude-opus-latest 或 claude-bridge/claude-sonnet-latest。

四、Context 優化（三層防護）

第一層：Agent 自檢

Heartbeat 觸發 自動壓縮

Heartbeat 時自動檢查 workspace 檔案大小：

# Agent heartbeat 自檢
for f in WORKLOG.md MEMORY.md CONTEXT.md; do
  SIZE=$(wc -c < "$f" 2>/dev/null || echo 0)
  echo "$f: ${SIZE} bytes"
done

超過預算則立即壓縮：

檔案 上限
WORKLOG.md < 2K
MEMORY.md < 3K
CONTEXT.md < 5K

第二層：Bridge 告警

WARNING

Prompt > 50K tokens 時，Bridge log 輸出 WARNING。

第三層：Session 壓縮

自動觸發

Session 接近 context 上限時觸發 memoryFlush + compact refresh。

五、清除污染 Session（400 錯誤處理）

Session 被污染（400 Bad Request）時，需清除三層：

# 1. Bridge state
rm ~/openclaw-claude-bridge/state.json

# 2. Claude CLI sessions
claude sessions list
# 手動刪除有問題的 session，或清除全部：
# claude sessions clear

# 3. OpenClaw sessions
# 在 OC 內執行 /new 建立新 session
清除順序：Bridge state → Claude CLI sessions → OpenClaw sessions。三層都清除後，系統即回復正常。

六、Dashboard

存取：http://localhost:3458/（或 http://<Mac-IP>:3458/）

功能一覽

  • 即時活動動態 — 思考、工具呼叫、刷新、完成、錯誤狀態即時顯示
  • Context 窗口使用量 — 綠色 <40%、琥珀色 40-65%、紅色 >65%
  • 請求表格 — 成本、耗時、快取命中率一覽
  • Session 清理 — 一鍵清除超過 24h 的 sessions

注意事項

請留意以下事項：
  • .env 改動後要重跑 install-mac.sh
  • Session 被 400 污染要清 3 層（bridge state + CLI sessions + OC sessions）
  • Bridge port 3456 只綁定 localhost，外部無法存取
  • --tools "" 已禁用 Claude 原生工具，安全無虞
