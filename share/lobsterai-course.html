✅ Skill 绑定 按需分配

❌ 不要一个 Agent 什么都做

03 System Prompt

写好人设，Agent 更 聪明

✏️ 角色定位

例：「你是一个___专家，擅长___」

📏 行为准则

例：「回答简洁 / 先分析再执行 / 遇到不确定的先问」

📋 输出格式

例：「用表格呈现 / 分步骤说明 / 中文回答」

🔒 安全边界

例：「不删除重要文件 / 操作前确认」

💡 使用技巧

✅ 推荐做法

把复杂任务 拆成小步骤

第一次用新功能，让龙虾 先解释方案

善用 Agent 预设，不同任务不同 Agent

重要操作加 「操作前先确认」

❌ 避免事项

不要 一次给太多指令

不要让 AI 操作你不了解的系统

不要把 敏感密码 直接写在对话中
