03 Agent 智能体

你的专属 AI 助手

Agent = 有人设和能力的 AI 助手

  • 📝 System Prompt — 定义角色和行为
  • 🧰 工具权限 — 能操作哪些东西
  • 📂 工作目录 — 工作的文件夹范围
  • 🔌 Skill 绑定 — 自动加载技能包

💡 可以创建多个 Agent：

写作助手 / 数据分析师 / 编程助手 / PPT 大师

设置 → Agent 配置

03 System Prompt

写好人设，Agent 更 聪明

✏️ 角色定位

例：「你是一个___专家，擅长___」

📏 行为准则

例：「回答简洁 / 先分析再执行 / 遇到不确定的先问」

📋 输出格式

例：「用表格呈现 / 分步骤说明 / 中文回答」

🔒 安全边界

例：「不删除重要文件 / 操作前确认」

💡 使用技巧

✅ 推荐做法

把复杂任务 拆成小步骤

第一次用新功能，让龙虾 先解释方案

善用 Agent 预设，不同任务不同 Agent

重要操作加 「操作前先确认」

❌ 避免事项

不要 一次给太多指令

不要让 AI 操作你不了解的系统

不要把 敏感密码 直接写在对话中

❓ 常见问题

Q: 龙虾会泄露我的数据吗？

A: 龙虾在本地运行，数据不出你的电脑。只有调用 LLM 时会发送对话内容。

Q: API 费用大概多少？

A: DeepSeek V3 非常便宜，日常使用每天几毛钱。

Q: 可以同时运行多个任务吗？

A: 可以！龙虾支持多个 Agent 并行工作。

Q: 断网了还能用吗？

A: 需要网络连接 LLM API。但可以用本地模型（Ollama）离线运行。
