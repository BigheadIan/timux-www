03 Agent 如何协作？

自动选择 + 互相调用

🤖 Agent 自动选择

1️⃣ 你对龙虾说「帮我写篇公众号文章」

2️⃣ 龙虾根据任务描述 → 自动匹配 Content Agent

3️⃣ Content Agent 需要素材 → 调用 Explorer Agent 搜索

4️⃣ 完成后交回总指挥 → 汇报结果

💡 关键机制

自动匹配

每个 Agent 有 description 描述自己能做什么。龙虾根据你的问题，自动选最合适的 Agent

Sub-Agent 调用

总指挥 Agent 可以把子任务分配给专业 Agent，像老板分工给部门主管

并行执行

多个 Agent 可以同时工作，不用排队等，效率翻倍

03 实战：创建 Agent

4️⃣ 选择模型

DeepSeek Reasoner — 内容创作需要深度思考

创建 Agent — 基础信息 + System Prompt

03 System Prompt

写好人设，Agent 更 聪明

✏️ 角色定位

例：「你是一个___专家，擅长___」

📏 行为准则

例：「回答简洁 / 先分析再执行 / 遇到不确定的先问」

📋 输出格式

例：「用表格呈现 / 分步骤说明 / 中文回答」

🔒 安全边界

例：「不删除重要文件 / 操作前确认」

💡 使用技巧

✅ 推荐做法

把复杂任务 拆成小步骤

第一次用新功能，让龙虾 先解释方案

善用 Agent 预设，不同任务不同 Agent

重要操作加 「操作前先确认」

❌ 避免事项

不要 一次给太多指令

不要让 AI 操作你不了解的系统

不要把 敏感密码 直接写在对话中
