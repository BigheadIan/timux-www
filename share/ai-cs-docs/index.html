从零开始搭建你的专属 AI 客服

第一部分：快速入门

1. AI训练 — 创建你的专属AI客服

核心功能：AI训练是系统的核心功能，让你通过简单的步骤训练出专属于你的行业AI客服。

操作步骤：

  1. 进入左侧菜单「AI训练」页面
  2. 点击「新增租户」，选择行业类型（旅游、餐饮、电商等16种），填写租户名称
  3. 创建成功后，系统自动生成3个基础意图（问候、转人工、未知问题）
  4. 点击「添加意图」创建自定义意图（如：价格咨询、退款流程等）
  5. 为每个意图添加对话范例，范例越多AI越准确
  6. 支持「智能导入」— 上传任意格式文档，AI自动解析为结构化对话

2. 渠道配置 — 连接你的通讯平台

核心功能：区域管理是配置渠道的地方，将AI客服连接到LINE等通讯平台。

操作步骤：

  1. 进入「区域管理」页面
  2. 点击「新增区域」或编辑现有区域
  3. 填写LINE Channel ID 和 Secret
  4. 选择AI训练集（绑定刚才创建的租户）
  5. 设置工作时间和图片选单
  6. 复制 Webhook URL 到 LINE Developers Console

3. 客服工作台 — 实时接管AI对话

核心功能：当AI无法回答或客户需要人工服务时，客服人员可以在工作台接管对话。

操作步骤：

  1. 进入「客服工作台」页面
  2. 左侧显示等待中的对话队列（按优先级排序）
  3. 点击任意对话即可接管，AI暂停回复
  4. 使用右侧罐头消息快速回复
  5. 对话结束后点击「关闭对话」并选择归类
